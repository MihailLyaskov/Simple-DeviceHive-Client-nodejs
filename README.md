# Simple-DeviceHive-Client-nodejs
Simple DeviceHive Client controlling ESP8266 gpio pins

## Installing NodeJS and NPM

```sh
sudo apt-get update
sudo apt-get install nodejs
sudo apt-get install npm
```
## Dependencies

```sh
npm install
```
## Run

```sh
node app.js
```
